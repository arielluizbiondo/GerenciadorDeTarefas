# 📋 Gerenciador de Tarefas em PHP

Este projeto é um **sistema simples de gerenciamento de tarefas**, desenvolvido em **PHP**, **HTML** e **CSS**.
Ele permite registrar tarefas, visualizar tarefas cadastradas e marcar tarefas como resolvidas.

---

# 🚀 Funcionalidades

✔ Registrar novas tarefas  
✔ Definir prioridade da tarefa  
✔ Visualizar lista de tarefas cadastradas  
✔ Marcar tarefas como **resolvidas**  
✔ Limpar todas as tarefas registradas
